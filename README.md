# WindTerm 繁體中文（台灣習慣用語）優化語系檔

本專案致力於為 [kingToolbox/WindTerm](https://github.com/kingToolbox/WindTerm)（最新 Release v2.7.0）提供更完整且符合台灣地區資訊用語習慣的繁體中文（L10n 本地化）語系檔。

> [!NOTE]
> **聲明：** 本專案為獨立之語系優化專案，旨在提供 WindTerm 繁體中文介面的本土化修正，並非官方的程式碼 Fork 庫。

---

## 💡 專案背景與改動說明

WindTerm 官方版本目前雖內建正體中文，但其語系檔部分內容因機器翻譯或直譯偏差，未能完全符合台灣的資訊專業用語習慣，對使用者的操作流暢度造成一定影響。

本專案針對官方 `zh-CN.locale`（包含 1,414 個翻譯項目）進行了全量的中文化本地化校正，主要優化項目包括：

### 1. 修正直譯與未本地化詞彙
為了提升介面的易讀性，本專案修正了多處直譯偏差，例如：
* **按鍵對照 (Keys)**：將直譯之「鑰匙 🔑」修正為符合軟體情境的「**按鍵** ⌨️」。
* **驗證機制 (Pageant)**：將直譯之「選美認證 👸」修正為 SSH 金鑰代理程式的「**Pageant 驗證** 🔒」。
* **游標與顯示 (Blink/Block/Bold)**：將「眨眼 👀」修正為「**閃爍** ✨」；「堵塞 🚫」修正為「**方塊** ⏹️」游標；「膽大 💪」修正為「**粗體** 🅰️」字型。
* **介面面板 (Bars)**：將「酒吧 🍺」修正為「**欄/列** 📊」介面元素。

### 2. 台灣地區資訊習慣用語校正
本專案全面將簡體中文或非台灣慣用的電腦用語，調整為台灣使用者習慣的標準用語：
* 「文件」 ➡️ 「**檔案**」（對應 File）。
* 「文件夾」 ➡️ 「**資料夾**」（對應 Folder）。
* 「會話」 ➡️ 「**工作階段**」 或 「**連線**」（對應 Session）。
* 「網絡」 ➡️ 「**網路**」。
* 「本地」 / 「遠程」 ➡️ 「**本機**」 / 「**遠端**」。
* 「配置」 / 「保存」 ➡️ 「**設定**」 / 「**儲存**」。
* 「窗口」 / 「鼠標」 / 「快捷鍵」 ➡️ 「**視窗**」 / 「**滑鼠**」 / 「**快速鍵**」。
* 「菜單」 / 「菜單欄」 / 「狀態欄」 ➡️ 「**選單**」 / 「**選單列**」 / 「**狀態列**」。
* 「標籤頁」 / 「標籤」 ➡️ 「**分頁**」（對應 Tab）。
* 「重新加載」 / 「撤消」 ➡️ 「**重新載入**」 / 「**復原**」。
* 「運行」 / 「調試」 ➡️ 「**執行**」 / 「**偵錯**」。
* 「波特率」 / 「數據位」 ➡️ 「**鮑率**」 / 「**資料位元**」。
* 「數據包」 ➡️ 「**封包**」。

## 🚀 搭配官方 Release 軟體套用教學

如果您使用的是 [WindTerm 官方 Release 版本](https://github.com/kingToolbox/WindTerm/releases)，您可以直接下載本專案的語系檔進行替換，無須下載整套軟體：

### 📥 步驟 1：下載語系檔
* 您可以直接點擊 [zh-TW.locale](https://raw.githubusercontent.com/inno-roywu/WindTerm-zh-TW/main/zh-TW.locale)（按右鍵選擇「另存連結為...」）。
* 或者，前往本儲存庫的 [Releases 頁面](https://github.com/inno-roywu/WindTerm-zh-TW/releases) 下載最新發佈的 `zh-TW.locale` 檔案。

### 📂 步驟 2：覆蓋官方語系檔
將下載的 `zh-TW.locale` 檔案放入您 WindTerm 安裝目錄下的 `global/locales` 資料夾中（覆蓋原本的同名語系檔）。
* 預設路徑通常為：`WindTerm_2.7.0/global/locales/zh-TW.locale`

### ⚙️ 步驟 3：於 GUI 介面套用
1. 啟動（或重新啟動）WindTerm。
2. 點擊頂部選單的 **`Settings` (設定)** -> **`Language` (語言)**。
3. 選擇 **`Chinese (Traditional) - 正體中文 (繁體)`**。
4. 介面將會立刻套用已優化的繁體中文！
